English
=======

Description
***********
Complete French Translation for MultiMerch for admin (Backend) and user (Frontend / shop)
No translation software involved. 
All files are UTF-8 w/o BOM encoded

WARNING: No translation for admin and user OpenCart Core. You must download before the « French Pack Language OC2 » by MLeroux for the OpenCart Core.

Installation
************

To install this language package, extract the zipped package locally
and transfer the files (same structure as you can see) with FTP
to your server (current shop installation).

Install first the « French Pack Language OC2 » by MLeroux and follow the install details.

********************
This extension is created under best effort.
Unlikely we cannot guarantee for any lost of data or malfunction.

Français
=======

Description
***********
Traduction complète pour MultiMerch pour l’espace admin (Backend) et pour l’utilisateur.
Tous les fichiers sont codés UTF-8 w/o BOM

ATTENTION : pas de traduction pour OpenCart incluse dans ce fichier. Vous devez d’abord télécharger le ficher « French Pack Language OC2 » par MLeroux pour OpenCart.

************

Pour installer cette traduction, il suffit d’extraire les fichiers de l’archive zip puis de les télécharger sur votre serveur en gardant la même structure.

Installer d’abord le « French Pack Language OC2 » de MLeroux et suivez l’installation.
